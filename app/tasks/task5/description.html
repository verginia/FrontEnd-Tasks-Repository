HTML Terms

HTML:
Stands for Hyper Text Markup Language for creating web pages.
HTML element:
An individual component of an HTML document, consisting of an opening tag, content, and a closing tag, if applicable.
HTML tag:
Are the keywords on a web page that define how your web browser must format and display your web page. It consists of angle brackets surrounding the tag name.
Tags can be opening tags, closing tags, or self-closing tags.
HTML attribute:
Additional information provided within the opening tag of an HTML element to modify its behavior or provide extra information about the element.
Examples: href, src, alt. style, lang, title, width, height.
HTML comment:
A portion of the HTML code that is not displayed in the web browser but can be used to add notes or comments for developers to understand the code.
HTML object element:
An HTML element used to embed external resources, such as images, audio, video, or other multimedia content, into an HTML document.
HTML special character:
Characters that have a special meaning in HTML and must be represented by character entities or escaped sequences to display them correctly in the browser.
Examples include < for less than (<), > for greater than (>), and & for ampersand (&).
